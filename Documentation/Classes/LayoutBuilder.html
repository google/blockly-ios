  • buildLayoutTree(forWorkspaceLayout:)

    Builds the entire layout tree for self.workspaceLayout based on the current model (ie. self.workspaceLayout.workspace).

    Throws

    BlocklyError: Thrown if the layout tree could not be created for this workspace.

    Note

    To increase performance during initialization, this should only be called after the entire workspace model has been constructed.

    Swift

  • buildLayoutTree(forBlock:engine:)

    Builds a BlockLayout tree for a given block and assigns it to the block’s delegate property. This includes all connected blocks.

    Throws

    BlocklyError: Thrown if the layout could not be created for any of the block’s inputs.

    Declaration

    engine

    Return Value

    The associated layout for the block.
